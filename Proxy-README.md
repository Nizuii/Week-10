# What is a Proxy Server.

- Retrives data on the internet on behalf of the user.
- It means basically a proxy server acts as a middle man between the client and the web server.
- It provides privacy that means it allows you to surf the internet anonymously because it hides your IP address.
- Without using proxy your public IP address is visible.
- But with a proxy, your public IP address is not visible because as the proxy server is retriving the web page for you only the IP address of the proxy server can be seen viewing the web page.
