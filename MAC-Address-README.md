# What is MAC Address

- MAC Address stands for **Media Access Control Address**.
- It is an identifier that every network device uses to uniquely identify itself on a network.
- No 2 device in the world wont be having same MAC address.
- MAC address is made up of 6 byte hexa decimal number that is burned into every NIC.
